# CIS 2336 – Web Development Demo Project

Welcome to the demo repository for **CIS 2336 – Web Development**.

This repository contains the example code used during class demonstrations. We will gradually build a complete web application by adding new features and concepts each week.

## What You Will Use This Repository For

- Follow along during class
- Practice coding examples
- Review demo code after class
- Experiment with HTML, CSS, JavaScript, and server-side code
- Prepare for assignments and projects

## Required Software

Before using this repository, make sure you have:

- [Visual Studio Code](https://code.visualstudio.com/)
- [Git](https://git-scm.com/)
- A GitHub account

## How to Clone This Repository

1. Open **Visual Studio Code**.
2. Open the terminal:

   **Terminal → New Terminal**

3. Run this command:

```bash
git clone https://github.com/HebAly/cis2336-summer2026-demo.git
```
## How to Open the Project

After cloning, go into the project folder:

```bash
cd cis2336-summer-2026-demo
```
Then open it in VS Code:

1. Open VS Code.
2. Select File → Open Folder.
3. Open the cis2336-summer2026-demo folder.

## How to Get Weekly Updates

Before each class, open the terminal inside the project folder and run:
```bash
git pull
```
This downloads the newest demo code.

## How to Open HTML Files

You can open HTML files in two ways.

- Option 1: Open Directly

Double-click the .html file.

- Option 2: Use Live Server

In VS Code:

- Install the Live Server extension.
- Open an HTML file.
- Right-click inside the file.
- Select Open with Live Server.

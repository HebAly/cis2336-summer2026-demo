# Module 2 – HTML and CSS

In this module, we build a simple personal portfolio website and then style it using CSS.

Folder Overview:

| Folder | Purpose |
|----------|----------|
| 01-html-starter | Starter files for the HTML activity |
| 02-html-complete | Completed HTML solution |
| 03-css-starter | Starter files for the CSS activity |
| 04-css-complete | Completed HTML and CSS solution |

Recommended Workflow:

1. Complete the HTML activity using `01-html-starter`.
2. Compare your work with `02-html-complete`.
3. Complete the CSS activity using `03-css-starter`.
4. Compare your work with `04-css-complete`.

Happy coding!

# CSS Starter

This folder contains the completed HTML website and a starter CSS file.

Your task is to style the website using CSS by following the in-class demonstration and activity instructions.

Files:

- `index.html` – Home page
- `hobbies.html` – Hobbies page
- `style.css` – CSS file where styles will be added
- `person.png` – Image used in the website

Work in this folder during the CSS activity.
